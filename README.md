# AI Pedagogy — Smart Learning System

AI-powered pedagogy and analogy recommendation system that runs entirely on your local machine.

## 🚀 Quick Start

To start the local AI server, always run this command in your terminal:

```bash
python app.py
```

### 🌐 Accessing the App
Once the server says `[OK] Model loaded and ready!`, open your browser to:
**[http://localhost:5000](http://localhost:5000)**

---

## ✨ Features
- **AI Lesson Generation**: Personalized pedagogy, explanations, analogies, and visualizations.
- **Interactive Diagrams**: Mermaid-powered flowcharts with zoom and pan support.
- **AI Quick Quiz**: Test your knowledge immediately after any lesson.
- **Premium Analytics**: Track your progress with animated Chart.js dashboards.
- **Hardware Optimization**: Built for 12-core CPUs with model pre-warming for instant inference.

## 🛠 Troubleshooting
If you see any "Windows Error" or encoding issues, the current `app.py` has been specifically patched to handle them. Just restart the command.
